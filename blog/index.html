About This Project

I've been on and off of Weight Watchers for many years. It has not cured me for my love of dessert. So this is an exercise to try and find the tastiest brownies, for the lowest number of points possible.

Note: I did not say healthiest. I'm not opposed to using any tricks I can to optimize for low number of points, which I have!
